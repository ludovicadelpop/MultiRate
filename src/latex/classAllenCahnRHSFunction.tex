\subsection{Detailed Description}
Class for RHS of PDE $ \frac{\partial u}{ \partial t} = \sigma * \frac{\partial^{2}u}{\partial x^{2} + u(1-(u^{2}))} \frac{\partial u(x0,t)}{ \partial x} = 0; \frac{\partial u(xL,t)}{ \partial x} = 0; $
